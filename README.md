# FORM-BASED

Form-based is a Web App that allows user to create, customize and compile forms.

## TECHNOLOGIES

### FRONTEND

- [React](https://reactjs.org)
- [Vite](https://vitejs.dev)
- [Material UI](https://mui.com)

### BACKEND

-[Laravel](https://laravel.com)
- mySQL


## INSTALLATION
1. Clone the Repo
2. install dependecies
```bash
npm install
```

```bash
npm run dev

```
